# Patient Readmission Analytics Dashboard

## Project Overview
This project analyzes hospital patient readmissions to identify patterns, trends, and factors contributing to readmissions.

The dashboard provides insights into:
- Total Patients
- Readmission Rate
- Department-wise Analysis
- Age Group Analysis
- Length of Stay Analysis
- Monthly Readmission Trends

## Tools Used
- Power BI
- SQL
- Microsoft Excel

## Key Features
- Interactive dashboard
- KPI cards
- Trend analysis
- Department performance comparison
- Readmission pattern identification


## Insights
- Certain departments showed higher readmission rates.
- Readmission frequency varied across age groups.
- Length of stay had a noticeable impact on readmission probability.
